#  Real-Time Chat Application

##  Prodigy Infotech Internship - Task 04

##  Project Overview
A real-time chat application using WebSocket technology enabling instant messaging between users with chat rooms and private conversations.

##  Technologies Used
### Frontend
- React.js
- Socket.io-client
- CSS3

### Backend
- Node.js
- Express.js
- Socket.io
- MongoDB & Mongoose
- bcryptjs
- JSON Web Tokens (JWT)

##  Features
-  Real-Time Messaging
-  Chat Rooms
-  User Authentication
-  Chat History
-  Typing Indicators
-  Online Users Count
-  Create New Rooms
-  JWT Authentication
